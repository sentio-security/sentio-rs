01 · language

Why port to Rust?

  • 01
    TS hit a ceiling

    Fine for a prototype. Need of reducing the false positive with AST based parsing.

  • 02
    Need a real AST + one binary

    syn for Anchor/Rust parsing · speed · ship as a single CLI on crates.io.

  • 03
    Result: full Rust stack

    sentio-core + sentio-cli. No dual TS/Rust stack. No source upload.
